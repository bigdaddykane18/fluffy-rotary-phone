Informationen zu Domänennamen finden

Ein Auszug aus Small, Sharp Software Tools von Brian P. Hogan

Wenn Sie eine Anfrage an facebook.com stellen, sucht Ihr Computer nach dem Domänennamen, um diesen in eine IP-Adresse zu übersetzen, damit Ihr Computer diese Verbindung herstellen kann. Eine IP-Adresse verweist auf einen Computer im Internet, ein Domänenname ist jedoch leichter zu merken als eine Zahlengruppe. Domain Name System (DNS)-Server speichern die Zuordnungen von IP-Adressen zusammen mit anderen Informationen. Es gibt mehrere Befehlszeilentools, mit denen Sie diese Server abfragen können.

Verwenden Sie den digBefehl, um die Domänennameneinträge zu durchsuchen und die damit verbundenen IP-Adressen zu finden. Dies ist eine zuverlässige Möglichkeit, Informationen über eine Domain zu erhalten:

Manchmal müssen Sie herausfinden, wem ein Domainname gehört, damit Sie ihn kontaktieren können. Oder Sie möchten prüfen, ob ein Domainname existiert, damit Sie ihn erwerben können. Einige skrupellose Unternehmen bieten Online-Domain-Suchen an. Wenn Sie nach einer gewünschten Domain suchen, kaufen sie diese automatisch, sodass sie Ihnen beim Bezahlen mehr berechnen können.

Mit dem whoisBefehl können Sie schnell feststellen, ob ein Domänenname vorhanden ist. Wenn es vorhanden ist, können Sie zusammen mit den Kontaktinformationen sehen, wem es gehört. Der whoisBefehl verwendet Domänennamenserver, um die Registrierungsinformationen einer Domäne abzufragen.

Unter Ubuntu müssen Sie den whoisBefehl möglicherweise installieren mit apt:

​ ​$ ​​sudo​​ ​​apt​​ ​​install​​ ​​whois​


​ ​$ ​​whois​​ ​​pragprog.com​

Basierend auf diesen Informationen:

  1. An wen würden Sie sich bei technischen Problemen wenden?
  2. Wann wurde die Domain erstmals registriert?
  3. Warum sind Ihrer Meinung nach so viele Nameserver miteinander verbunden?

Manchmal kennen Sie den Domainnamen nicht. Manchmal haben Sie nur die IP-Adresse. Der hostBefehl erleichtert das Nachschlagen eines Domänennamens anhand einer IP. Wenn Sie Sicherheitsprotokolle durchsehen und Treffer von einer bestimmten IP-Adresse erhalten, versuchen Sie, mit dem hostBefehl den Domänennamen der Anfrage nachzuschlagen:

​ ​$ ​​host​​ ​​8.8.8.8​


​ 8.8.8.8.in-addr.arpa domain name pointer google-public-dns-a.google.com.
